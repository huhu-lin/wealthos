### 3. KDJ 指標計算

**評級：⚠️ PARTIAL ISSUE - 待確認**

#### 問題描述

```javascript
// 第 159-171 行：calcKDJ()
const slice = closes.slice(i-period+1, i+1);
const high = Math.max(...slice);
const low = Math.min(...slice);
const rsv = high===low ? 50 : (closes[i]-low)/(high-low)*100;
```

**問題點：**

1. **RSV 基礎的歧義**
   - 當前實現：基於期間內收盤價的最高/最低
   - 標準 KDJ：應基於期間內燭台的最高價(high)/最低價(low)
   
2. **實際影響**
   - 在價格上下波動但收盤價變化不大的情況下，會高估 RSV
   - 例如：某天開盤高於前期最低，收盤低於前期最低，真實 low 會更低
   - 影響程度：中等（特別是高波動率市場）

3. **資料可用性**
   ```javascript
   // K 線資料結構（見 line 221）
   data.map(d => ({ time:d.date, open:d.open, high:d.high, low:d.low, close:d.close }))
   ```

**待確認原始意圖後再修復。** 可能的設計選擇：
- 選項 A：保持現狀（基於收盤價，簡化實現，便於回測對比）

function calcAnnualizedStats(equityValues, tradingDays) {
  // 1. 年化報酬率 = (末值/初值)^(252/交易日數) - 1
  const annReturn = Math.pow(1 + totalReturn, 252 / tradingDays) - 1;
  
  // 2. 年化波動率 = sqrt(日報酬方差 × 252)
  const annVol = Math.sqrt(variance * 252);
  
  // 3. 夏普比率 = (年化報酬 - 2% 無風險率) / 年化波動率

  const winRate = (winDays / totalDays) * 100;
}

- ✅ 夏普比率：(年化報酬 - 無風險率) / 年化波動率（標準定義，假設無風險率 2%）
- ✅ 勝率：正報酬日數百分比（直觀的策略穩定性指標）

## 附錄：計算公式驗證

### 年化報酬率
```
公式：(末值/初值)^(252/交易日數) - 1
例：100萬初值 → 110萬末值，250交易日
  = (110/100)^(252/250) - 1
  = (1.1)^1.008 - 1
  ≈ 10.13% 年化報酬
```

### 夏普比率
```
公式：(年化報酬 - 無風險率) / 年化波動率
例：年化報酬 10%，年化波動率 15%，無風險率 2%
  = (0.10 - 0.02) / 0.15
  = 0.533
解釋：每承受 1% 波動，獲得 0.533% 超額報酬
```

### 勝率
```
公式：正報酬日數 / 總交易日數 × 100%
例：250個交易日中 160 個正報酬
  = 160/250 × 100%
  = 64% 勝率
```
